MTE.js
webアプリケーションガジェット

MTE.jsとは

MTE(MouseTrackinngEffect)はマウスカーソルの軌道上にエフェクトを発生させることができるプログラムです。
HTML上にスクリプトリンクとして記述することで、簡単に実装させることができます。
※ MTE.js は開発中のガジェットです。

使い方

ベース

HTMLに以下のコードを書くことで、埋め込むことができます。 
<script src="https://ao2324-00.github.io/mte/MTE.js">
      オプション
</script>

オプション

  • type エフェクトの形
  • color エフェクトの色
  • size エフェクトの大きさ
  • range エフェクトの表示数
  • opacity エフェクトの透明度
  • translate エフェクトの平行移動
  • rotate　エフェクトの回転
  • transition エフェクトのアニメーション
  • render エフェクトの表示方法

type エフェクトの形

type="タイプ名"

エフェクトの形を指定します(デフォルトはcircle)。
square : 四角形, circle : 円, rhomb　: ひし形, star : 星型, dog : 足跡, cat : 足跡

color エフェクトの色

color="色"

エフェクトの色を指定します(デフォルトはblue)。
指定方法は、
色名, RGB, RGBA, グラデーション などがあります。

size エフェクトの大きさ

size="大きさ"

range エフェクトの表示数

range="数値"

opacity エフェクトの透明度

opacity="数値"

translate エフェクトの平行移動

translate="(x, y)"

rotate　エフェクトの回転

rotate="角度"
rotate="follow"

transition エフェクトのアニメーション

transition="(種類, 遅れ, 変化時間)"

render エフェクトの表示方法

render="drag"

サンプルプログラム

サンンプル1

<script src="https://ao2324-00.github.io/mte/MTE.js">
    type="star"
    color="rgb(20, 90, 255)"
    size="6-30px"
    range="100"
    opacity="0.5"
</script>

サンンプル2

<script src="https://ao2324-00.github.io/mte/MTE.js">
    type="circle"
    color="radial-gradient(orange, red)"
    size="4px,8px,12px"
    range="1000"
    render="drag"
</script>

サンンプル3

<script src="https://ao2324-00.github.io/mte/MTE.js">
    type="cat"
    color="rgb(90,50,0)"
    size="5px, 0px, 5px, 0px"
    range="700"
    opacity="0.5"
    translate="(20, 0), (0, 0), (-20, 0), (0, 0)"
    rotate="follow"
    transition="fade, 0.7, 1"
</script>
