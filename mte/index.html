MTE.jsとは

MTE(MouseTrackinngEffect)はマウスカーソルの軌道上にエフェクトを発生させることができるプログラムです。
HTML上にスクリプトリンクとして記述することで、簡単に実装させることができます。
※ MTE.js は開発中の為、仕様が変更される可能性があります。

使い方

ベース

HTMLに以下のコードを書くことで、埋め込むことができます。 
<script src="https://ao2324-00.github.io/mte/MTE.js">
      オプション
</script>

オプション

  • type エフェクトの形
  • color エフェクトの色
  • size エフェクトの大きさ
  • range エフェクトの表示数
  • opacity エフェクトの透明度
  • translate エフェクトの平行移動
  • rotate　エフェクトの回転
  • transition エフェクトのアニメーション
  • render エフェクトの表示方法

type エフェクトの形

type="タイプ名"

エフェクトの形を指定します(デフォルトはcircle)。
"square" : 四角形, "circle" : 円, "rhomb" : ひし形, "star" : 星型, "dog" : 足跡, "cat" : 足跡

color エフェクトの色

color="色"

エフェクトの色を指定します(デフォルトはblue)。
指定方法は、
色名, RGB, RGBA, グラデーション などを使うことが出来ます。

size エフェクトの大きさ

size="大きさ"

エフェクトの大きさを指定します(デフォルトは10px)。
サイズはpx以外の単位でも指定可能です。
サイズを","で区切ることで複数のサイズを左から順に繰り返すことができます。
また、"数値-数値px"のように"-"で区切ると、その範囲でランダムなサイズのエフェクトを表示させる事ができます。

range エフェクトの表示数

range="数値"

エフェクトを何個まで表示させるかを指定します(デフォルトは50個)。
1000個までの表示は問題なく可能ですが、四角形、円形以外のエフェクトはこれ以上表示すると描画が遅れる可能性があります。
また、transitionを使用しても描画が少し遅れる可能性があります。

opacity エフェクトの透明度

opacity="数値"

エフェクトの透明度を指定します。
数値は1.0(不透明)〜0.0(透明)で使用します。

translate エフェクトの平行移動

translate="(x, y)"

エフェクトの表示位置をずらすために使用します。
"(x1, y1), (x2, y2)"のように","で区切って指定すると、左から順の繰り返しでずらす事がきます。
rotateと併用すると、rotateが必ず先に適用されます。よって、指定角度に比例してx、y軸も角度が変化します。

rotate　エフェクトの回転

rotate="角度"
rotate="プロパティ"

表示されるエフェクトの角度を指定します。
プロパティに"follow"を指定すると、マウスカーソルがある方を常に上とした角度でエフェクトが表示されます。
translateと併用すると、rotateが先に適用されます。

transition エフェクトのアニメーション

transition="(種類, 数値1, 数値2)"

エフェクトが表示されてからのアニメーションを指定します。
種類 : アニメーションの種類を指定します。
数値1: 何秒後にアニメーションを開始するか。
数値2: 何秒かけてアニメーションするか

  • フェードアウト : "(fade, 数値1, 数値2)"

render エフェクトの表示方法

render="プロパティ"

エフェクトを表示するイベントを指定します。
"drag" : ドラッグ中に表示します。

サンプルプログラム

サンンプル1

<script src="https://ao2324-00.github.io/mte/MTE.js">
    type="star"
    color="rgb(20, 90, 255)"
    size="6-30px"
    range="100"
    opacity="0.5"
</script>

サンンプル2

<script src="https://ao2324-00.github.io/mte/MTE.js">
    type="circle"
    color="radial-gradient(orange, red)"
    size="4px,8px,12px"
    range="1000"
    render="drag"
</script>

サンンプル3

<script src="https://ao2324-00.github.io/mte/MTE.js">
    type="cat"
    color="rgb(90,50,0)"
    size="15px, 0px, 15px, 0px"
    range="500"
    opacity="0.5"
    translate="(30, 0), (0, 0), (-30, 0), (0, 0)"
    rotate="follow"
    transition="fade, 0.7, 1"
</script>
